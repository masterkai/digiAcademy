Windows Server 2016基礎系列建置之二：系統管理進階實務
課程介紹
透過本課程的學習，將幫助學員認識本機使用者和群組、NTFS檔案權限、使用者設定檔的管理，以及磁碟 管理作業。
學習重點
  • 本機使用者與群組帳戶管理
  • 淺談本機NTFS檔案權限安全性管理
  • 管理使用者設定檔之工作環境
  • 磁碟管理
學習大綱
  • 第一堂：本機使用者與群組帳戶管理
  • 第二堂：淺談本機NTFS檔案權限安全性管理
  • 第三堂：管理使用者設定檔之工作環境
  • 第四堂：磁碟管理
軟體版本
2016
課程長度
194分

  • 想學習Windows Server 2008的網路服務建置與系統管理的人員
